# AirForShare clone text & files sharing web app based on your geolocation (100 meters)

## Features

- Text sharing (100 meters radius)
- Files sharing upto 1gb (100 meters radius)
- Responsive
- Drag and drop files
- Remove files
- Auto remove files (after 5 minutes)
- Copy text
- Clear text
- Attractive UI

## Frontend

- Typescript
- Reactjs (vite)
- SCSS
- Axios
- Momentjs
- React icons

## Backend

- Javascript
- Nodejs
- Expressjs
- Mongodb (mongoose)
- Multer
- Momentjs
- Nodemon
- Morgan
- Cors
